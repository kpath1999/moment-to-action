# Moment-to-Action: Low-Power, Multimodal AI-Enabled Wearable Systems

This project aims to enable Moment-to-Action capabilities on wearable devices. Our system will dynamically adapt sensing, model inference, and compute resources to balance accuracy, latency, and energy constraints. By utilizing heterogeneous platforms like the Qualcomm QCS6490, we aim to bridge the gap between high-performance AI requirements and the strict energy budgets of wearable hardware, ensuring sensitive data is efficiently analyzed locally and privately.
